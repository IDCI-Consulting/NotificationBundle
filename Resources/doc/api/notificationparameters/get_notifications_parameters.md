IDCINotificationBundle API: [GET] Notifications Parameters
==========================================================

List all notifications parameters

## General
|             | Values
|-------------|-------
| **Method**  | GET
| **Path**    | /notificationsparameters
| **Formats** | json
| **Secured** | false

## HTTP Response codes
| Code | Description
|------|------------
| 200  | Ok
| 404  | Not found
| 500  | Server error
